# Lux Render Landing

Landing estática de alto nivel para Lux Render.

## Incluye
- `index.html`: estructura completa.
- `styles.css`: diseño responsive mobile/desktop, estética premium dark/tech.
- `app.js`: interacciones simples.
- `assets/`: logo Lux Render y foto de Esteban si estaban disponibles.

## Enfoque comercial
Posiciona a Lux Render como estudio de IA aplicada, marketing visual, sistemas de captación, landings y agentes IA omnicanal.

## Cómo abrir
Abrir `index.html` directamente en navegador.

- Cambiar el CTA final al WhatsApp/formulario real.
- Integrar video o capturas reales de Conserje Neural.
- Ajustar precios/planes según cierre comercial.
